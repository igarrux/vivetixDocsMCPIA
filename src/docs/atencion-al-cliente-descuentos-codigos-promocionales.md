# Crear un descuento o código promocional para tu evento

**URL fuente:** https://vivetix.com/f/p/pf/descuentos-codigos-promocionales  
**Categoría:** Atención al cliente / FAQ

---

¿Alguna vez has querido premiar a alguno de tus asistentes con un descuento? ¿O has querido hacer una campaña puntual a un grupo de asistentes interesantes?

Si la respuesta es "sí", te interesará saber cómo se pueden hacer descuentos a tus asistentes con la plataforma de Vivetix.

## Cómo crear un cupón de descuento

Una vez creado el evento, en el panel de administración que te aparecerá en tu evento, puedes pulsar en **más opciones → cupones**.

Una vez dentro de cupones te aparecerá el cuadro de configuración donde podrás completar los datos que necesites.

## Campos del cupón

- **Código:** Se refiere a números o letras que los usuarios tendrán que rellenar. Por ejemplo puede ser: `Promo10`. Si dejas el campo en blanco se generará el código aleatorio.

- **Descuento:** Es el porcentaje que se le descontará al comprador al aplicar el cupón. Por ejemplo 10%. Este campo es obligatorio y puede ser un descuento de 0 al 100%.

- **Fecha de vencimiento:** Es la fecha en la que el cupón caducará. Por ejemplo 01-06-2021. Si se deja vacío no habrá caducidad.

- **Número de usos:** Es la cantidad de usos que puede tener el cupón. Por ejemplo 20 si quieres que solo aplique a 20 tickets. Si un usuario hace una compra de 30 tickets, el descuento se le aplicará a los 20 primeros. Si no se rellena este campo serán usos infinitos.

- **Gasto mínimo:** Si quieres poner una cantidad mínima para que aplique el cupón. Por ejemplo 20€: de esta forma si un usuario hace una compra de 10€ no se le aplicará el cupón, en cambio si la hace de 30€ se le aplicará. Al dejarlo vacío no habrá gasto mínimo.

- **Tipo de ticket:** Esta opción sirve para limitar el descuento a una opción en particular. Por ejemplo "entrada general", de tal forma que el descuento solo aplique a esa entrada pero no al resto. Si se deja sin seleccionar, el descuento aplicará a todos los tickets.

## Importante

Ten cuidado con los códigos generados. Si el código es público no hay problema, pero si quieres que el código sea para un grupo reducido de personas o si el descuento es un buen descuento, es mejor utilizar un código aleatorio. Si usas por ejemplo "promo" o varios códigos "promo1", "promo2", es posible que sean fácilmente adivinables y los puedan llegar a adivinar otras personas.
