# ¿Qué ofrece Vivetix para organizadores?

**URL fuente:** https://vivetix.com/f/p/pf/que-ofrece-vivetix-para-organizadores  
**Categoría:** Sobre nosotros / FAQ

---

## Antes del evento

- Listas de asistentes exportables e imprimibles
- Cobrar con tarjeta a los asistentes — pasarela de pagos o TPV para eventos
- Hacer preguntas por escrito a tus asistentes
- Añadir asistentes manualmente sin pasar por el proceso de compra (por ejemplo para pagos externos)
- Descargarte la lista de asistencia
- Si se agotaran los tickets o entradas, se activará una lista de espera
- Si se cerrara la lista definitivamente, se activará una lista para que puedas avisar de los siguientes eventos a los interesados
- Si alguien compra el ticket o entrada para otros podrás tener el destinatario
- Podrás elegir si se permiten las devoluciones

## Al empezar el evento o durante

- Control de asistencia por listas impresas o escaneando códigos QR con el móvil
- Seguridad para evitar tickets duplicados

## Después del evento

- Tener la lista de asistentes y no asistentes
- Para los que llegan tarde y no pudieron adquirir el ticket o entrada se activará una lista donde se podrán apuntar y avisar para el próximo evento

## En cualquier momento

- Ponerte en contacto con tus asistentes
- Accesible tanto desde el ordenador como el móvil
- Fácil de usar
- Un alto porcentaje de usuarios que se convierten en compradores
